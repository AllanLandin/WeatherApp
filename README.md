# App do Tempo! ⛈️
A aplicação permite a consulta do tempo atual de qualquer cidade do planeta.
# Tecnologias 💻
Utiliza React, SASS, HTML e as informações fornecidas pela [OpenWeatherAPI](https://openweathermap.org/)
